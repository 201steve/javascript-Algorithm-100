# 🧾 문제

```javascript
# 문제7 : 변수명

다음 중 변수명으로 사용할 수 없는 것 2개를 고르시오.

1)  age
2)  Age
3)  let
4)  _age
5)  1age

```

# 👨🏻‍🏫 답

```javascript
3) let
5) 1age
```

# 🖥 내가 한 풀이

```javascript
* let은 자바스크립트 예약어
* 변수는 숫자로 시작하면 안된다
```

# 🎯 어차피 찾아볼꺼 잘 써놓자

```javascript
예약어는 변수로 사용할 수가 없다
변수명의 시작은 (_)밑줄, ($)달러기호, 문자열로 시작한다
```
